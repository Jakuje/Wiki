h1. ACOS5

ACOS5 cards are not (yet) supported by OpenSC but work is ongoing. Development can be followed in "https://github.com/pacew/OpenSC/tree/acos5":https://github.com/pacew/OpenSC/tree/acos5 ("development notes wiki":https://github.com/pacew/OpenSC/wiki)

Note: the acos5 branch seems to be dead now. It has not seen any new commits since 2011, and has never been merged into main development branch. Hoewer ACS offers prepackaged PC/SC drivers for it's CryptoMate tokens for download. Any information regarding the supported features are welcome.
