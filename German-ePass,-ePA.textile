h1. German ePass, ePA

<b>TODO: Merge/generalize to a generic "ICAO MRTD":http://www.icao.int/security/mrtd/Pages/default.aspx page</b>

Germany already uses a passport with builtin contactless smart card (ePass = elektronischer Reisepass). Germany plans to introduce an identification card with similar functionality (ePA = elektronischer Personalausweis) in 2010.

If you have informations about ePass or ePA, please let us know or add a link to the list below:

Information about the german ePass / ePA:

 * "informations about german passports from the Bundesinnenministerium":http://www.bmi.bund.de/cln_144/sid_9EE07FCA0CF9E2CCD37C327651A54B1D/DE/Themen/Sicherheit/PaesseAusweise/paesseausweise_node.html
 * "das eCard-API-Framework (BSI TR-03112)":https://www.bsi.bund.de/cln_174/ContentBSI/Publikationen/TechnischeRichtlinien/tr03112/index_htm.html
 * "short info about ePA":http://www.bmi.bund.de/cae/servlet/contentblob/122652/publicationFile/15722/elPersonalausweis_InfoGrafik.pdf
 * "ePA field-test, call for participants":http://www.cio.bund.de/cln_093/DE/IT-Projekte/Leuchtturmprojekt_ePA/Anwendungstest_ePA/anwendungstest_node.html
 * "JMRTD - software to read RFID chips in biometric e-passports":http://jmrtd.org/
 * "OpenPACE":http://sourceforge.net/projects/openpace/ OpenSSL implementation of the Password Authenticated Connection Establishment (PACE) used with the ePA
 * "Virtual Smartcard Architecture":http://sourceforge.net/projects/vsmartcard/ includes an emulation of an ePass and an emulation of a smart card reader, which is capable of establishing a secure connection with the ePA using PACE
